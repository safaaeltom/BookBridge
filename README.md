BookBridge
BookBridge is a web-based platform that allows parents to give away used school books and helps other families access free educational materials.
The goal of this project is to reduce book waste and improve access to education by creating a simple and user-friendly book exchange system.

Features
- Browse books by country
- Responsive design
- API integration

Tech Stack
- React
- JavaScript
- CSS

Future Improvements
- Additional features
